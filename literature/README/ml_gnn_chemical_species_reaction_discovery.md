# Literature Search 1: GNN and Machine Learning for Discovering Chemical Species, Reactions, and Reaction Networks

Search date: 2026-06-17

## Scope

This list focuses on machine learning methods that help discover, predict, or enumerate chemical reactions, reaction products, reaction centers, retrosynthetic steps, reaction barriers, bond-breaking events, and reaction networks. Some papers use graph neural networks directly; others use sequence models, neural-symbolic planning, ML potentials, or ML-assisted automated reaction exploration.

## How This Relates To Combustion Mechanisms

These papers are mostly from organic chemistry, catalysis, and quantum chemistry, but they are relevant to combustion because they address the same building blocks needed for mechanism generation:

- What species/intermediates should be considered?
- Which bonds can break or form?
- Which reactions are likely?
- What are the products?
- What are the barriers and rate constants?
- How can a reaction network be explored automatically?

For combustion, the most directly useful connections are reaction-network exploration, rate estimation, bond dissociation prediction, and automated mechanism generation.
